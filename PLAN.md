# Pixel Agents — 개선 내역

## 1. OpenRouter 에이전트 지원 추가

Claude Code CLI 외에 OpenRouter API를 사용하는 에이전트도 Pixel Office에 캐릭터로 등장하도록 지원을 추가했다.

- `node dist/hooks/openrouter-agent.js` 실행 시 캐릭터가 생성됨
- 대화를 주고받는 동안 캐릭터가 타이핑 애니메이션을 보여줌
- 에이전트 종료 시 캐릭터가 사라짐
- API 키는 `~/.pixel-agents/openrouter-config.json`에 저장 (소스 코드에 하드코딩 금지)

---

## 2. 새로고침 후 캐릭터가 사라지는 버그 수정

브라우저를 새로고침하면 이미 실행 중이던 에이전트의 캐릭터가 모두 사라지는 문제를 수정했다.

- **이전**: 새로고침 시 캐릭터가 전부 사라짐
- **이후**: 새로고침 후에도 실행 중인 에이전트 캐릭터가 그대로 유지됨

---

## 3. 에이전트 종료 후 캐릭터가 남는 버그 수정 (OpenRouter)

`openrouter-agent.js`를 종료해도 캐릭터가 화면에 그대로 남아있는 문제를 수정했다.

- **Ctrl+C / `/exit` 로 종료 시**: 캐릭터 즉시 사라짐
- **터미널 강제 종료(창 닫기) 시**: 캐릭터 즉시 사라짐
- **`kill -9` 강제 종료 시**: 3분 후 자동으로 사라짐 (운영체제 수준의 강제 종료라 즉시 처리 불가)

---

## 4. Pixel Agents 서버보다 먼저 실행된 Claude 세션 미감지 버그 수정

Pixel Agents 서버가 켜지기 전부터 이미 실행 중이던 Claude Code 세션이 캐릭터로 나타나지 않는 문제를 수정했다.

- **이전**: 서버 시작 후 새로 실행한 Claude 세션만 감지됨
- **이후**: 서버 시작 전부터 실행 중이던 세션도 약 6초 내에 자동 감지되어 캐릭터로 등장함

---

## 5. 오피스 레이아웃 변경 — 아늑한 홈 오피스 스타일

기존의 밋밋한 사무실 배치에서 Stardew Valley 스타일의 아늑한 공간으로 레이아웃을 재구성했다.

**왼쪽 방 (따뜻한 나무 바닥):**

- PC와 의자가 있는 작업 데스크 4개 배치 (2×2 배열)
- 데스크 사이사이에 식물과 선인장 배치

**오른쪽 방 (밝은 바닥):**

- 소파 U자 배치 + 소파 테이블로 아늑한 라운지 공간 구성
- 큰 화분과 다양한 식물 배치
- 하단에 소파와 소파 테이블 추가

**벽 장식:**

- 양쪽 방에 책장, 그림, 행잉 플랜트, 시계 배치

---

## 6. 지시 즉시 캐릭터 활성화 개선

사용자가 메시지를 입력하고 보내도 캐릭터가 idle 상태를 유지하다가, 실제로 도구를 사용하기 시작할 때 뒤늦게 활성화되는 문제를 개선했다.

- **이전**: 메시지 전송 후 수 초의 딜레이가 지난 뒤 캐릭터가 활성화됨
- **이후**: 메시지를 보내는 즉시 캐릭터가 "작업 중" 상태로 전환됨
